## 🚀 快速部署

### 方法一：Vercel 一键部署（推荐）

1. 访问 [vercel.com](https://vercel.com)
2. 登录（可用 GitHub 账号）
3. 点击 "New Project"
4. 导入本项目的 GitHub 仓库
5. 点击 "Deploy"
6. 完成！获得链接：`https://your-app.vercel.app`

### 方法二：本地测试

```bash
# 安装 Node.js（如果未安装）
# 访问 https://nodejs.org 下载安装

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 方法三：直接打开（最简单）

直接用浏览器打开 `index.html` 即可体验！

## 🛠️ 扩展内容

### 添加新章节

1. 在 `storyData` 对象中添加新章节：

```javascript
const storyData = {
    // ...现有章节...
    "4a": {
        "title": "第 4 章 XXX",
        "content": `剧情内容...`,
        "choices": [
            {"text": "选择 1", "next": "5a"},
            {"text": "选择 2", "next": "5b"}
        ]
    }
};
```

2. 确保 `next` 指向正确的下一章 ID

### AI 生成新内容

使用以下提示词让 AI 生成新章节：

```
请写第 X 章的互动小说内容，包括：
- 章节标题
- 2000 字左右的剧情
- 3-5 个选择分支
- 每个分支的下一章 ID

故事背景：[简述前情提要]
前章选择：[用户选择了什么]
```
